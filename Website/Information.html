Information

Common Terms & Their Meaning (8086/8088 instruction set of Intel)

Instruction Example Meaning
mov mov ebx, 110 ebx = 110, Assigns Register ebx the Value 110
add add eax, 200 eax = eax + 200, Increments the value of eax by 200, can also be used on pointers
sub sub eax, 200 eax = eax - 200, Decrement the value of eax by 200, can also be used on pointers
inc inc eax eax = eax + 1, Increments the value of eax by 1, can also be used on pointers
dec dec eax eax = eax - 1, Decrements the value of eax by 1, can also be used on pointers
and and eax, ecx eax = eax & ecx, Logical and on eax and ecx, stored in eax
or or eax, ecx eax = eax || ecx, Logical or on eax and ecx, stored in eax
xor xor eax, ecx eax = eax xor ecx, Logical xor on eax and ecx, stored in eax
not not eax eax = ~(eax), Logical not on eax, stored in eax
