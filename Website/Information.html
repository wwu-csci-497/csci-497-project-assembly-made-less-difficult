Common Instructions & Their Meaning (8086/8088 instruction set of Intel)

Instruction Example Meaning
mov mov ebx, 110 ebx = 110, Assigns Register ebx the Value 110
cmp cmp eax, 60 60 - eax, this result is stored in a special register that is used for jump conditions
jmp label jmp .section_two Start reading code from the labeled section .section_two
je / jz je .section_two / jz .section_two Jump to .section_two if equal / jump to section_two if 0
jne / jnz jne .section_two / jnz .section_two Jump to .section_two if not equal / jump to .section_two if not 0
js js .section_two Jump to .section_two if less than 0
jns jns .section_two Jump to .section_two if not less than 0
jg / jnle jg .section_two / jnle .section_two Jump to .section_two if greater than 0 / Jump to .section_two if not less than or equal to 0
jge / jnl jge .section_two / jnl .section_two Jump to .section_two if greater than 0 or equal to 0 / Jumpt to .section_two if not less than 0
jl / jnge jl .section_two / jnge .section_two Jump to .section_two if less than 0 / Jump to .section_two if not greater than 0 and not equal to 0
jle / jng jle .section_two / jng .section_two Jump to .section_two if less or equal to 0 / jump to .section_two if not greater than 0
cmove / cmovz cmove eax, rax / cmovz eax, rax eax = rax if equal / eax = rax if 0
cmovne / cmovnz cmovne eax, rax / cmovnz eax, rax eax = rax if not equal / eax = rax if not 0
add add eax, 200 eax = eax + 200, Increments the value of eax by 200, can also be used on pointers
sub sub eax, 200 eax = eax - 200, Decrement the value of eax by 200, can also be used on pointers
inc inc eax eax = eax + 1, Increments the value of eax by 1, can also be used on pointers
dec dec eax eax = eax - 1, Decrements the value of eax by 1, can also be used on pointers
and and eax, ecx eax = eax & ecx, Logical and on eax and ecx, stored in eax
or or eax, ecx eax = eax || ecx, Logical or on eax and ecx, stored in eax
xor xor eax, ecx eax = eax xor ecx, Logical xor on eax and ecx, stored in eax
not not eax eax = ~(eax), Logical not on eax, stored in eax
lea lea eax, [ebx+8] eax = [ebx+8], Load Effect Address
pop pop eax eax = (value that was on top of the stack)
push push 0x16 The top of the stack now equals 0x16
ret ret pops the return address off of the stack and returns control to that location
sal sal eax, 4 eax = eax*(2^4), shifts left 4 binary places
sar sar eax, 4 eax = eax/(2^4), shifts right 4 binary places
test test eax, rax In a special address used by jmp and other command eax & rax is stored
